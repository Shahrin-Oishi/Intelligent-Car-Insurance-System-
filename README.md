# 🚗 Intelligent Car Insurance System 🚗

## 📌 About Project
This is a simple web project that calculates car insurance cost based on vehicle type, age of car and safety features.



## How it works
User enters:
- Owner name
- Vehicle type
- Car age
- Safety option

Then the system calculates insurance premium using a simple formula.


## Formula Used
Premium = Base Rate × Vehicle Type × Age Factor × Safety Factor

- Base Rate = 1000
- Vehicle Type changes risk (Sedan, SUV etc.)
- Older cars have higher risk
- Safety features reduce cost



## Technologies Used
- HTML
- CSS
- JavaScript

To practice basic web development and simple logic building using JavaScript.
